TO UPDATE USING ANSIBLE 2.4.1.0:

    Make sure you can log into Crystal device or VM to be updated without password using ssh.
    To do this do:
    ssh-copy-id service@SOME_VEC_HOSTNAME

    Verify you can log in without password using:
    ssh service@SOME_VEC_HOSTNAME

    add a group containing the SOME_VEC_HOSTNAME to /etc/ansible/hosts

    ansible <host-pattern> --module-name=proxy-web-setup.yml

    Example:

    ansible-playbook --limit=SOME_VEC_HOSTNAME --user=service --ask-pass --become-method=su --ask-become-pass --syntax-check proxy-web-setup.yml
    ansible-playbook --limit=SOME_VEC_HOSTNAME --user=service --ask-pass --become-method=su --ask-become-pass                proxy-web-setup.yml
